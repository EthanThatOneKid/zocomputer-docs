
Work with the user's connected Google Sheets spreadsheets.

## Parameters

<ParamField type="string">
  Exact tool name returned by `list_app_tools`.
</ParamField>

<ParamField type="object">
  The configured props for the action (e.g., `{"spreadsheetId": "abc123", "sheetName": "Sheet1"}`).
</ParamField>

<ParamField type="string">
  Optional email address to select a specific connected account.
</ParamField>
