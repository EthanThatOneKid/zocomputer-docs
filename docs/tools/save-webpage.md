
Save webpage content to the user's Articles directory.

Supports YouTube URLs by saving a transcript-based article (video embed + transcript). For standard webpages, the content is saved as Markdown. For file URLs (PDFs, images, audio, video files, etc), the file is downloaded. This tool uses [Zo's browser](https://docs.zocomputer.com/browser) to visit the page and extract clean article content.

## Parameters

<ParamField type="string">
  URL of the webpage to save.
</ParamField>
