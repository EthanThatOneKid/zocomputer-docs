
Work with the user's connected Google Calendar account.

## Parameters

<ParamField type="string">
  Exact tool name returned by `list_app_tools`.
</ParamField>

<ParamField type="object">
  The configured props for the action (e.g., `{"calendarId": "primary", "singleEvents": true, "orderBy": "startTime", "timeMin": "2024-01-01T00:00:00-05:00", "timeMax": "2024-01-02T00:00:00-05:00"}`)
</ParamField>

<ParamField type="string">
  Optional email address to select a specific connected account.
</ParamField>
