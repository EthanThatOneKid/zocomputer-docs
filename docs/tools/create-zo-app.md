
Create a Zo App: a standalone app served on its own subdomain.

## Parameters

<ParamField type="string">
  Display name for the app, e.g. 'My Cool App'. Used to derive a slug when one isn't given.
</ParamField>

<ParamField type="string">
  Optional subdomain label (the 'acme' in acme.zo.app). Lowercase alphanumeric + hyphens, 3-63 chars. Derived from name if omitted.
</ParamField>
